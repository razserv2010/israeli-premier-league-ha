# 🇮🇱 Israeli Premier League — Home Assistant Integration

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)

אינטגרציה ל-Home Assistant שמושכת אוטומטית משחקי **ליגת העל הישראלית** ל-3 הימים הקרובים.

## ✨ יכולות
- 📅 משחקים ל-3 ימים קרובים
- ⏰ שעות בשעון ישראל
- 📺 ערוצי שידור
- 🔄 עדכון אוטומטי כל 3 שעות
- 📊 תוצאות חיות במהלך משחקים
- 🏟️ שם האצטדיון

## 📦 התקנה דרך HACS
1. HACS → Integrations → ⋮ → Custom repositories
2. הוסף: `razserv2010/israeli-premier-league-ha` | Category: `Integration`
3. חפש **Israeli Premier League** והתקן
4. הפעל מחדש את Home Assistant

## 🔑 API Key
1. הירשם ב-[api-sports.io](https://api-sports.io) — חינמי (100 בקשות/יום)
2. Dashboard → My Account → API Key

## ⚙️ הגדרה
Settings → Devices & Services → Add Integration → Israeli Premier League → הזן API Key
